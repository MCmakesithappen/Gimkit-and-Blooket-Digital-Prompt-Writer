Saving to Google Sheets & Downloading as CSV

This is the most reliable way to create a CSV that both Gimkit and Blooket will accept every time.

  • 1Go to sheets.google.com and open a new blank spreadsheet
  • 2In your AI tool, copy the table of questions the AI returned — select all of it from the header row to the last question
  • 3Click cell A1 in your Google Sheet and paste — the questions will fill in automatically across the columns
  • 4Make sure the first row is the header row with column names — do not delete it
  • 5Click File in the top menu → Download → Comma Separated Values (.csv)
  • 6Your CSV file will download to your computer — it is now ready to upload
💡 Pro tip: When pasting into Google Sheets, if the text lands in one column instead of spreading across columns, use Edit → Paste Special → Paste as plain text first, then try again. Google Sheets should automatically detect the commas and split the data into the correct columns.

⚡ Upload to Gimkit

  1. Go to gimkit.com → Create Kit
  2. Click Import → CSV
  3. Upload your downloaded CSV file
  4. Map columns if prompted → Import!

📚 Upload to Blooket

  1. Go to blooket.com → Create Set
  2. Select Spreadsheet Import
  3. Click Upload CSV
  4. Select your file → Save Set!

ℹ️ About This Tool
🎯

NC Standards Aligned

Every prompt is built around the NC Standard Course of Study for your selected grade and content area, including optional standard number and description.

📊

Mixed Difficulty Levels

Questions automatically spread across ~40% Recall (L1), ~40% Application (L2), and ~20% Analysis (L3) — each labeled in the question text.

🎲

Randomized Answers

The AI is instructed to shuffle answer positions so the correct answer never always appears in the same column across questions.

📥

Google Sheets → CSV

Pasting into Google Sheets and downloading as CSV guarantees a perfectly formatted file that Gimkit and Blooket will accept every time.
