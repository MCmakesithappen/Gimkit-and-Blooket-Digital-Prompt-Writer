ℹ️ About This Tool
🎯

NC Standards Aligned

Every prompt is built around the NC Standard Course of Study for your selected grade and content area, including optional standard number and description.

📊

Mixed Difficulty Levels

Questions are automatically spread across ~40% Recall (L1), ~40% Application (L2), and ~20% Analysis (L3) — each labeled in the question text.

🎲

Randomized Answers

The AI is instructed to shuffle answer positions so the correct answer never always appears in the same column.

📥

Instant CSV Download

Paste the AI's CSV output and download a properly named .csv file — the only format accepted for bulk import by Gimkit and Blooket.
